1: first pull the image

docker pull docker.elastic.co/elasticsearch/elasticsearch:8.12.1

2: after pulling create the container as 

docker run -d --name elasticsearch \
  -p 9200:9200 -p 9300:9300 \
  -e "discovery.type=single-node" \
  -e "xpack.security.enabled=false" \
  docker.elastic.co/elasticsearch/elasticsearch:8.12.1

ðŸ”¹ -p 9200:9200: Maps the container port to localhost
ðŸ”¹ discovery.type=single-node: Tells ES to run in single-node mode
ðŸ”¹ xpack.security.enabled=false: Disables security (so you can use it without auth for now)

now do "curl http://localhost:9200"
or visit it in the browser
